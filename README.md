# Selenium Test Automation Project
This repository contains automated UI test scenarios developed using **Selenium WebDriver** and **TestNG**.
The project is designed to simulate real-world test automation practices using a structured and maintainable framework.

## Features
* Automated login test scenarios (valid & invalid cases)
* Page Object Model (POM) design pattern
* Reusable utility methods
* Maven-based project structure
* Basic assertions and validations

## Project Structure
* `pages` → Page Object classes
* `tests` → Test cases
* `utilities` → Driver management & reusable methods
* `pom.xml` → Project dependencies

## How to Run Tests
```bash
mvn clean test
```
Or run tests directly via TestNG in IntelliJ.

## Test Approach
* Page Object Model (POM) for maintainability
* Separation of concerns (test vs page logic)
* Reusable components for scalability
